# Green Industrial Consulting Website

موقع شركة خبراء التطوير الصناعي والاستشارات الخضراء، مبني باستخدام Next.js.

## التشغيل المحلي

```bash
npm install
npm run dev
```

ثم افتح:

```text
http://127.0.0.1:3000
```

## أوامر مهمة

```bash
npm run build
npm run start
```

## صفحات العملاء

- `/clients/login` صفحة دخول العملاء.
- `/clients/dashboard` لوحة متابعة العميل.

## ملاحظات الرفع

- ملفات البناء مثل `.next` لا يتم رفعها.
- ملفات الطلبات المحلية داخل `storage` لا يتم رفعها لأنها قد تحتوي بيانات عملاء.
- ملف الصور المضغوط `public/images/img.zip` لا يتم رفعه لأن الصور المستخرجة موجودة داخل المشروع.
